DevConnect

DevConnect est une plateforme sociale innovante conçue spécifiquement pour les développeurs. Elle permet aux programmeurs de partager leur code, collaborer sur des projets et échanger en temps réel via un système de chat intégré.

Fonctionnalités principales :

  • Chat en temps réel avec support de code
  • Éditeur de code collaboratif
  • Système de partage de snippets
  • Profils développeurs personnalisables
  • Système de matching par compétences
  • Intégration GitHub

Technologies utilisées :

Vue.js 3 Firebase Realtime DB WebSocket Monaco Editor GitHub API Socket.io
