Weather Hub

Weather Hub est une application météo moderne qui fournit des prévisions météorologiques en temps réel avec une interface utilisateur élégante et interactive. L'application utilise des animations fluides pour afficher les changements météorologiques et offre des prévisions détaillées sur 7 jours.

Fonctionnalités principales :

  • Prévisions météo en temps réel
  • Visualisation sur 7 jours
  • Animations météorologiques dynamiques
  • Géolocalisation automatique
  • Mode sombre/clair adaptatif

Technologies utilisées :

JavaScript ES6+ API OpenWeather HTML5 Canvas CSS3 Animations LocalStorage Geolocation API
